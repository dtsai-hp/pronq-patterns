Welcome to the Pronq Patterns Library

This pattern library has been built to help you navigate your way through the Pronq brand. We provide colors, fonts, and user interface elements that can used on web pages, marketing materials, and applications that must follow the Pronq brand.

Tools and Tricks

Keep an eye out for these three buttons throughout the pattern library. Each will give you more of what you need to help you use our elements and incorporate our brand into your work.
Detailed Specs
For designers you will see this button under sections. Click this to see details on what this element should look like in different states.
Code
For developers, we have which will provide the code needed to include this element in a page.
Assets
Some of our elements have assets that you can download that are meant to be used with them. These can also be found on the Downloads page.
Instructions
We have built out quick instructions on how to place some of our elements into your projects. These can also be found on the Downloads page.

Pages

Welcome This page has information on how to use our pattern library, as well as information on our brand. You can get an understanding of the message behind Pronq.
Styles Under styles you will find colors, typography, and information on the grid we use on Pronq.com. These are the very basic elements of our style guide.
UI This section has features such as buttons, forms, sliders, and other user interface elements that are used often throughout the site.
Components The components has larger elements such as the global navigation, menus, tabs, carousels, etc. These are built using our UI elements and are structured to be responsive.
Templates Under templates you will find page specific templates for the product details pages, pricing pages, and several others.
Resources Under resources you can find our logo, color palette, fonts, and other assets you may need We have included a library of .psd files of some of our components that can dropped into your designs as well as links to other resources you may need.
Change Log This pattern library is a work in progress and we are constantly adding to and improving it. You can see what changes we have made in the change log.
